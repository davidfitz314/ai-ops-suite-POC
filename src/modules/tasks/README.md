# Tasks Module

## What it does

Tracks, organizes, and manages all work items generated by the system.

````

---

## Workflow

```mermaid
flowchart TD
    A[Task Created] --> B[Open State]
    B --> C[Start Task]
    C --> D[In Progress]
    D --> E[Complete Task]
    E --> F[Done]
    F --> G[Close Task]
```

---

## UI Mapping

```mermaid
flowchart LR
    List[Task List] --> Detail[Task Detail Panel]

    List --> Grouped[Grouped by Status]

    Detail --> Title[Editable Title]
    Detail --> Status[Status Controls]
    Detail --> Actions[Task Actions]
```

---

## Purpose

```
Acts as the system of record for all actionable work.

```

---

## Notes

* Global state managed via TaskContext
* Tasks are grouped by status:

  * open
  * inProgress
  * done
  * closed
* Updates are immediate (no save required)
* New tasks are auto-selected upon creation

````
